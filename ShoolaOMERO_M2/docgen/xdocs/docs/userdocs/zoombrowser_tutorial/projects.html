The project list can be used to identify datasets included in a project and to select a project for further investigation. Initially, all project names are printed in simple black font. Mousing over a project name selects it temporarily. The moused-over project name is printed in blue and slightly increased in size. All of the datasets contained in that project are highlighted in orange, and all of the other projects that contain any of those datasets are also highlighted. This provides an overview of the contents of each project, and the overlap between this project and other projects.

When the desired project has been identified, it can be selected with a single mouse click. When this happens, the dataset display is updated to show only those datasets found in the selected project.

 
Fig 2-2: The project label has been clicked on. This selects the project and updates the dataset view to show only those datasets found in the selected project.

Selected projects can be deselected by selecting another project, or by double-clicking on the selected project label.
