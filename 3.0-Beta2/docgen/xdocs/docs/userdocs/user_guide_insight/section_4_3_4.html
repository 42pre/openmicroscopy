To remove a project or dataset:

  1. In the left-hand pane of the Data Manager, select the project or dataset to be removed.
  2. In the Edit menu, select Remove (or Remove from current project for a dataset). This removes the selected project or dataset.

NB
A dataset is only removed from a project - it is not deleted from the system. The ability to recall existing datasets that have been removed will be added to future releases of OMERO.insight.
