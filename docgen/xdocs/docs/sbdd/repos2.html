The whole Shoola source code and other related artifacts can be freely downloaded (refer to Legal for licensing issues) from our CVS repository. To do so, you will need a CVS client. If you don't have it, you can get one from http://www.cvshome.org/. Before you start downloading from CVS, make a new directory and move there. Then, type the following at the command line prompt:

    cvs -d ':pserver:anoncvs@cvs.openmicroscopy.org.uk:/ome' login

When prompted for a password, enter anoncvs. Then, type:

    cvs -z3 -d ':pserver:anoncvs@cvs.openmicroscopy.org.uk:/ome' checkout Shoola

The above will download the whole Shoola module in the current directory. What you get is the latest version of the files on CVS. If you're interested in a specific branch/tag instead, then type:

    cvs -d ':pserver:anoncvs@cvs.openmicroscopy.org.uk:/ome' -r tag_id Shoola

Where tag_id is the label identifier for the branch/tag you're interested in, for example OME_2_2.
