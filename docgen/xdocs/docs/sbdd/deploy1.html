In order to run Shoola, your system must have:

  • A processor of 1GHz or faster.
  • A minimum of 256MB of RAM. (We strongly recommend 512MB.)
  • A network link of 100Mb (possibly dedicated) between your machine and the OME server site you intend to connect to.
  • A Java Runtime Environment, 1.4 or later.

 

Note
In principle, the software should run on any Java Runtime Environment. However, we've only been testing the system extensively on Linux/Blackdown JVM, Mac OS X/Apple JVM, Windows/Sun JVM.
