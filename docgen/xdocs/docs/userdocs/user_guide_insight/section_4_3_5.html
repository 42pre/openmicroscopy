You can add and view additional information to a selected project or dataset. For a selected project or dataset, the right-hand pane of the Data Manager will show information. The default view is to show Properties - i.e. the name and description, as shown in Figure 6 (for a project) and Figure 8 (for a dataset).

However you can also view Permissions information for the selected project or dataset, by selecting the Permissions tab (see Figure 9).
 
Figure 9: Viewing Permissions information for a selected project
If the right-hand pane appears blank when a project or dataset is selected, select the toolbar icon to view properties and permissions information in the right-hand pane.
