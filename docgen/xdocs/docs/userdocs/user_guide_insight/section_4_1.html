When you start OMERO.insight, you will be asked to log in (see Figure 3). Specify your user name and password (see the OMERO.admin User Guide for details of setting up user accounts).

You will also need to provide the path name of the OMERO server you will be accessing by selecting the button and entering the details in the form provided. Note that if you are running the OMERO Server locally on your own machine, enter 'localhost' as the server name.
