You can save a copy of the image in its currently displayed form, in a number of different image formats.

  1. Open the Controls menu and select the Save As option.
  2. Here you can:
    • Specify the name of the saved image file.
    • Select the destination of the saved image file.
    • Select the saved image file format from a choice of BMP, JPEG, TIFF or PNG.
    • Remember the destination folder you save the image to for future use.
