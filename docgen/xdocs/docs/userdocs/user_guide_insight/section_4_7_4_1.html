To categorise an individual image:

  1. Select the image filename.
  2. Then:
    • Either in the Edit menu, select Categorise.
    • Or select the toolbar button and then Categorise from the pop-up menu that appears.
  3. In the right-hand pane, you will initially see a list of available Category Groups. Select the appropriate Category Group and expand it by selecting the Expand button .
  4. Now, tick all categories you wish to add the selected image to. (Select All and Deselect All buttons are available to speed up the task of selection).
  5. Once the desired categories have been selected, select Save to confirm the categorisation.
