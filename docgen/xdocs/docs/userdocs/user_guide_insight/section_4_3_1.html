To create a new project:

  1. Either:
    • In the File menu, select Create New project.
    • Or select the icon.
  2. In the window that appears (see Figure 5), enter the name of the new project. You can optionally add a textual description of the project.
  3. Select Save to confirm.
  4. Your project should now be created (Figure 6 shows a new project created and named DemoProject).
