In the left-hand pane you might see projects listed but not their datasets (Figure 11). The number in square brackets after the project name is the number of datasets belonging to the project.

To see all datasets belonging to a project, click the Expand icon .

All datasets associated with the selected project will now be shown (see Figure 12). This icon will change appearance to to show that the project has been expanded to show datasets - clicking the icon again will collapse the display to show just the project again. Note that in Figure 11, the AnotherDemoProject image has no datasets - so cannot be expanded.
