OMERO is a set of software tools that supports image data management, visualisation and annotation. OMERO tools include:

  • an OMERO Server;
  • and a series of client applications:
    • OMERO.admin;
    • OMERO.importer;
    • OMERO.insight.

This User Guide concentrates on OMERO.insight, but the following information in this section gives a brief outline of the principles and concepts used by OMERO.
