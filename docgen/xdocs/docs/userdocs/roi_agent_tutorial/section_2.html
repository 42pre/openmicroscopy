We're now going to define a cylinder within the stack of the first time-point (t = 0).

  1. We create a new ROI by pressing the New ROI button and by entering the name of the ROI in the ROI Editor. This new ROI will be automatically given an id of ROI #0.
  2. Then we press the Oval tool button in the Drawing context panel.
  3. We now draw an ellipse on the displayed image -- the XY-plane at (z = 9, t = 0).
  4. In order to define a cylinder, we need to propagate this shape in the stack. Instead of drawing manually the same shape across the stack, we use the ROI Assistant to automate this task.
  5. In the ROI Assistant, we select the cell at (z = 9, t = 0), which corresponds to the XY-plane at (z = 9, t = 0). A little circle inside this cell indicates that this plane contains a 2D-selection. We now drag it down to (z = 5, t = 0). When the mouse is released, we select the fill-to-here option in the pop-up menu brought up on screen. New circles will appear to reflect that our initial 2D-selection has now been applied to all z between 9 and 5 at t = 0.
  6. The next step is to propagate the cylinder across time. Again, we do this using the ROI Assistant. We want to propagate from t = 0 to t = 6, so we enter these values in the From t and To t text fields, respectively (press enter each time). Then we press the Copy stack button . Again, the grid will be updated accordingly.

 

Tip
Selecting Show label in the Drawing context panel, will display the ROI id on screen.

By dragging the Z or T slider in the Viewer, it's possible to see where each 2D-selection is positioned within each XY-plane.

In a similar way we define the next ROI. This is going to be a cube that propagates across time. We first define a new ROI by pressing the New ROI button . The new ROI will be automatically given an id of ROI #1. At this point we press the Rectangle tool button and we draw a square on plane (z = 9, t = 0). In order to constraint the dimensions of the rectangle to be the same, we hold the Shift key as we draw. We need to look at this selection a little bit closer, so we bring up the ROI Inspector and we zoom our selection. We realize that some unwanted pixels ended up in our selection, so we resize it to cut them off the selection. Then in a similar way as explained above (steps 3 to 6), we define a cube and we propagate it across time.
