This tutorial will demo some of the key features of the ROI Analysis Tool. We'll show you how to:

  • Define ROIs.
  • Analyze those ROIs (min, max, mean, standard deviation, area, sum).
  • View the results of the analysis.
  • Manipulate those results.
  • Export the results.

 

For this tutorial, we are going to use an image containing several z-sections and time-points. After opening the image in the Viewer, we press the ROI Analysis button . The ROI Definition Tool appears on screen.
