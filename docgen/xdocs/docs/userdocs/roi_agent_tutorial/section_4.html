By default, the results are presented in a spreadsheet-like table. However, it's also possible to plot them in a graph. To do so, we press the Plot button in the ROI Results dialog and, in the Plot Results dialog, we select the results we want to plot.
Here we want to see how the max value varies across time for z = 5 and z = 6. We select the max value in the list box and the time-points as X-axis. We then select all time-points and the desired z-sections -- 5 and 6.

Tip
Each curve will be assigned a default color. Another color can be chosen by clicking on the colored square next to checkbox in the Plot Results dialog. This will bring up the Color Chooser dialog.

To compute the graph, we press the Graph button .
