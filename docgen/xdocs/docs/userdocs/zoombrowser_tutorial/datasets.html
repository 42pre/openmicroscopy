The dataset display can be used to examine datasets and explore the relationships between datasets and projects.

When the mouse is moved over a dataset, the datasets is highlighted in orange. If the mouse is over an empty area in the dataset (in other words, if it is not over an image), the name of that dataset will be shown in a tool tip. The names of projects containing the datasets will be highlighted.

 
Fig 3-1: The mouse has been moved over a dataset, highlighting the dataset and labels for the project in which this dataset can be found. The tooltip displays the name of the dataset.

Clicking on a dataset will zoom the display in to focus on that dataset. To zoom out, Right-click on the dataset or click on the background.

 
Fig 3-2: Clicking on a dataset selects it and zooms the display to focus on it.
