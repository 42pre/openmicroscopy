To manage users, click on the Scientists button in the toolbar to display the list of existing users (see Figure 4.2-1). You can then perform various Actions:

  • add new user (see Section 4.3),
  • import many users from external source (see Section 4.4),
  • delete user (Section 4.5),
  • edit users (see Section 4.6).
